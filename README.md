# 7xuanlu — Claude Code plugins

Personal marketplace for [@7xuanlu](https://github.com/7xuanlu)'s Claude Code plugins. Each plugin lives in its own repository and is referenced here, pinned by commit `sha`.

## Install

```
/plugin marketplace add 7xuanlu/claude-plugins
/plugin install <plugin>@7xuanlu
```

## Plugins

| Plugin | Description | Source |
|--------|-------------|--------|
| `origin` | Personal AI memory layer (MCP memory + skills) | [7xuanlu/origin](https://github.com/7xuanlu/origin) · `plugin/` subdir |
| `boule` | Research-backed multi-LLM council with debiased judging | [7xuanlu/boule](https://github.com/7xuanlu/boule) |
| `ultrapowers` | Unattended SDD/TDD build harness | [7xuanlu/ultrapowers](https://github.com/7xuanlu/ultrapowers) |

Each entry is pinned to a specific commit. Bump the `sha` (and `ref` tag) when a plugin releases.
